### Import

To use this widget in your project, include the following import statement in your file.

```
import 'package:gluestack_ui/gluestack_ui.dart';
```

It inherits all the properties of Flutter's [Container](https://api.flutter.dev/flutter/widgets/Container-class.html) widget.

#### GSAccordionItem

Contains all the parts of a collapsible section. GSAccordionItem consists of GSAccordionTitle and GSAccordionContent which defines the title and content of GSAccordion widget.

#### GSAccordionTitle

It inherits all the properties of Flutter's [Text](https://api.flutter.dev/flutter/widgets/Text-class.html) widget.

#### GSAccordionContent

It inherits all the properties of Flutter's [Text](https://api.flutter.dev/flutter/widgets/Text-class.html) widget.

#### GSAccordionIcon

It inherits all the properties of Flutter [Icon](https://api.flutter.dev/flutter/widgets/Icon-class.html) widget.
